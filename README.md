# Pesquisa de Opinião - Atendimento ao Cliente

Projeto em Python desenvolvido para a atividade da Agenda 8.

## Objetivo
Coletar nome, idade e opinião de 50 entrevistados sobre o atendimento prestado.

## Opções da pesquisa
1 - EXCELENTE
2 - BOM
3 - RUIM

## Resultado exibido
- Quantidade de respostas EXCELENTE
- Quantidade de respostas RUIM
